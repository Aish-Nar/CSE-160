Aishwaroopa Narayanan — Assignment 3

Description - This project is a first-person 3D world built with WebGL using textured cubes and a perspective camera. It includes WASD movement, mouse look, Q/E rotation, a 32x32 map with walls of different heights, brick textures, block placing and removing, and a simple chase game where you find the golden treasure before the red golem catches you(never does).

All of this is my own work with the help of the professor's videos
