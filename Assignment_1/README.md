Description - This project is a WebGL painting app that lets users draw shapes on a canvas using points, triangles, and circles. It includes controls for color, size, segments, and clearing the canvas.

Drawing - My drawing is based on my initials A and N. I sketched it on my Ipad first and then recreated it with WebGL triangles on the webpage. The A makes a fox and N makes a bunny. 

Awesomeness - I added "Rainbow Mode", which lets the user paint with changing random colors. 

All of this is my own work with the help of the professor's videos and searching up animals to draw on google.
